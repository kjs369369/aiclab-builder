# 마케팅 진단보고서 샘플

## 요약
매주 월요일 SNS 광고 성과를 부서별 엑셀 시트에 모아 보고하는 업무. 데이터 수집·정리에 매주 4~5시간 소요.

## 추천 AI 도구
- **Google Sheets + Apps Script**: 광고 플랫폼 API에서 자동 수집
- **Notion AI**: 주간 요약 자동 생성
- **Make.com**: 데이터 흐름 오케스트레이션 (Slack 발송까지)

## 시간 절감 예측
월 16~20시간 → 월 2시간 (약 90% 절감)

## 자동화 흐름
1. 광고 플랫폼(Meta/Google/Naver)에서 일별 KPI 자동 수집
2. Google Sheets에 정규화하여 누적
3. Notion AI로 주간 요약 + 인사이트 작성
4. 월요일 오전 9시 Slack #marketing 채널 자동 발송

## 코치 코멘트
초기 1회 셋업이 가장 중요. 각 광고 플랫폼 API 키 발급과 Apps Script 권한 설정에 1~2일 투자하면 이후 사실상 무인 운영.
