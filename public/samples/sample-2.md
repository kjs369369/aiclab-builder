# HR 진단보고서 샘플

## 요약
신입사원 온보딩 시 동일 자료 안내·FAQ 응답을 매번 반복. 분기별 신입 10~15명 대응에 인사 담당자가 평균 30분/명을 소비.

## 추천 AI 도구
- **NotebookLM**: 사내 인사 규정·복지 가이드 통합 Q&A
- **Slack 봇 (GPT-4o-mini)**: #onboarding 채널 상시 응답
- **Notion**: 온보딩 체크리스트 템플릿

## 시간 절감 예측
분기 5~8시간 → 분기 1시간 (약 85% 절감)

## 자동화 흐름
1. 사내 인사 규정·복지·시설 가이드를 NotebookLM 노트북에 업로드
2. Slack 봇이 신입 질문을 받아 NotebookLM으로 답변 생성
3. 답변 출처 표시 + 인사팀 멘션 옵션 제공
4. 월별 질문 로그 분석 → FAQ 갱신

## 코치 코멘트
민감한 인사 정보(연봉/평가)는 봇에서 제외하고, 인사팀 직접 응대 영역으로 분리하는 가드레일 설계 필수.
